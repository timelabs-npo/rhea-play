# Rhea Play — Claude Context

## What this is

Standalone macOS SwiftUI app extracted from the Rhea monorepo (`rh.1`). 12-pane operations centre for the Rhea multi-model AI advisory system.

## Structure

```
rhea-play/
  Sources/
    PlayApp.swift          # App entry, PlayShell (pane switcher), all pane views, MenuBarView
  packages/
    RheaKit/               # Shared Swift package (also used by iOS app)
      Package.swift
      Sources/RheaKit/
        AppConfig.swift    # API base URL, UserDefaults migration
        RheaStore.swift    # ObservableObject: auth state, agent status
        RheaAPI.swift      # URLSession wrappers for all Tribunal endpoints
        AuthView.swift     # JWT login/signup gate
        DialogView.swift   # Tribunal debate UI
        GovernorView.swift # Token budget display
        TasksView.swift    # Task queue (claim/complete/block)
        PulseMonitorView.swift
        AtlasWebView.swift # WKWebView wrapping Three.js graph
        OfficeView.swift
        OpsView.swift
        Theme.swift        # Color palette, fonts
        ... (25 total)
  project.yml              # xcodegen spec — run `xcodegen generate` before building
  Info.plist               # Bundle metadata
  .gitignore               # Excludes *.xcodeproj (generated), build/, .build/
```

## Key facts

- **Entry point**: `Sources/PlayApp.swift` — `CommandCentreApp` (@main), `PlayShell` (root view)
- **Pane enum**: `PlayShell.Pane` — 13 cases, each with label/icon/shortcut
- **API layer**: `RheaKit/RheaAPI.swift` — all HTTP calls, configurable base URL
- **Auth**: JWT via `RheaStore`, stored in Keychain via `KeychainAccess`
- **Default API URL**: `localhost:8400` (overridable in Config pane / UserDefaults)
- **Bundle ID**: `com.rhea.play`
- **Deployment target**: macOS 14.0
- **Xcode project**: generated by `xcodegen generate` — never committed to git

## Build

```bash
xcodegen generate
xcodebuild -scheme RheaPlay -configuration Release build
```

## Monorepo origin

In the monorepo `rh.1`:
- `play/Sources/` → `Sources/`
- `play/project.yml` → `project.yml` (with `../packages/RheaKit` path updated to `packages/RheaKit`)
- `packages/RheaKit/` → `packages/RheaKit/`

## Dependencies

Local: `packages/RheaKit/`
Remote: Pow, GRDB.swift, swift-collections, KeychainAccess, swift-markdown-ui, Starscream

## Constraints

- Do not commit `*.xcodeproj` — regenerate with xcodegen
- Do not commit `.build/` or `build/` directories
- Do not include secrets, `.env` files, or API keys
- NDI support requires `libndi` v6.2.0 at `/usr/local/lib` — graceful degradation if absent
